## [특정 형질을 가지는 대장균 찾기]

>핵심 개념: SELECT, 비트 연산

### 문제 요약
 2번 형질을 보유하지 않고 1번이나 3번 형질을 보유하고 있는 대장균 개체의 수 출력

### 데이터 구조
| 컬럼명 | 설명 |
|------|------|
| id | 대장균 개체의 ID |
| parent_id | 부모 개체의 ID |
| size_of_colony | 개체의 크기 |
| differentiation | 분화된 날짜 |
| genotype | 개체의 형질(정수) |
---

### 접근 방식
1. 형질을 이진수로 나타냈을 때 가장 오른쪽 비트부터 순서대로 각 번호 형질 여부 표현함을 인지 (형질 보유시 1)
2. **해당 위치 비트가 켜져 있음을 확인**하기 위해 **& 연산자 필요** (& 연산 결과 형질 보유 시 0이 아님)
3. WHERE 조건식에서 1번 3번 보유 여부는 둘중 하난만 성립되어도 되므로 or 연산자 사용, 2번 미보유 여부는 항상 성립 되어야 하므로 and 연산자 사용
4. 조건식에서 필터링 된 결과를 세기 위해 COUNT() 함수 사용


### 시행착오
- 단순히 bin()을 이용해 이진수로 바꾼 후 like문을 이용한 패턴 매칭을 이용해 코드가 길어짐
- **해당 위치 비트가 켜져 있음**을 확인하기 위한 **& 연산자**를 숙지하지 못함

---

### 최종 쿼리
```sql
SELECT COUNT(*) AS COUNT
FROM ECOLI_DATA 
WHERE (GENOTYPE&1 <> 0
    OR GENOTYPE&3 <> 0)
    AND GENOTYPE&2 = 0;
