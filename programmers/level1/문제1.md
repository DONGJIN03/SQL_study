# [문제 제목]

## 문제 한 줄 요약
- 무엇을 계산 / 추출하는 문제인지 핵심만 정리

---

## 데이터 구조
- 사용 테이블: `users`, `orders`
- 주요 컬럼
  - `user_id` : 사용자 ID
  - `created_at` : 생성 일시
  - `status` : 상태 값

---

## 접근 방식
- 집계 기준을 먼저 정함 (user 단위)
- 조건 필터링은 WHERE에서 처리
- 결과에 필요한 컬럼만 SELECT

---

## 시행착오
- 처음에는 `WHERE`에서 집계 조건을 처리하려다 결과가 달라짐
- `HAVING` 사용이 필요하다는 점을 놓침

---

## 최종 쿼리
```sql
SELECT
    user_id,
    COUNT(*) AS order_cnt
FROM orders
WHERE status = 'completed'
GROUP BY user_id
HAVING COUNT(*) >= 2;
