# SQL Concepts

이 폴더는 SQL 문제 풀이 과정에서  
**자주 실수하거나 헷갈렸던 개념을 판단 기준 중심으로 정리**한 공간입니다.

문제별 오답이나 시행착오는 포함하지 않으며,  
개념 자체를 **다시 틀리지 않기 위한 기준**으로 정리합니다.

---

##  정리 대상 기준

다음에 해당하는 경우에만 concepts에 문서화합니다.

- 여러 문제에서 반복적으로 헷갈린 개념
- 선택 기준이 명확하지 않았던 SQL 문법
- 기억해두면 문제 해결 속도가 빨라지는 개념

---

##  작성 원칙

- 문제 맥락 설명 x
- 개인적인 시행착오 서술 x
- **판단 기준과 정답 구조 중심 o**

각 문서는 다음 요소를 포함합니다.

1. 개념 요약
2. 언제 이 개념을 선택해야 하는지
3. 헷갈리기 쉬운 비교 대상과의 차이
4. 기억용 한 줄 기준
